============================================================
  KETSY STORE — IMAGE SAVE GUIDE
  How to put your photos into the website
============================================================

Your website is already set up to load images automatically.
You just need to save each photo into the correct folder
with the exact filename shown below.

------------------------------------------------------------
FOLDER STRUCTURE (inside your KETSY STORE project folder)
------------------------------------------------------------

KETSY STORE (1)/
└── images/
    ├── tshirts/
    ├── longsleeves/
    ├── hoodies/
    ├── fullset/
    ├── shorts/       ← no photos yet (emoji placeholder used)
    └── joggers/      ← no photos yet (emoji placeholder used)

------------------------------------------------------------
STEP-BY-STEP: HOW TO SAVE EACH PHOTO
------------------------------------------------------------

1. Open the chat where you uploaded the photos.
2. Right-click each photo → "Save Image As..."
3. Navigate to the correct folder (see list below).
4. Save it with the EXACT filename shown (must match exactly).
