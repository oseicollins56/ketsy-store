====================================================
 KETSY STORE – GitHub Push Guide
====================================================

STEP 1: Install Git (if not already installed)
-----------------------------------------------
Download Git from: https://git-scm.com/downloads
After installing, open Terminal (Mac/Linux) or Git Bash (Windows).

Verify it's installed:
  git --version


STEP 2: Create a GitHub Account
--------------------------------
Go to: https://github.com
Sign up for a free account if you don't have one.


STEP 3: Create a New Repository on GitHub
------------------------------------------
1. Log in to GitHub
2. Click the "+" icon at the top right → "New repository"
3. Repository name: ketsy-store
4. Description: Ketsy Store – Fashion & Custom Gifts E-Commerce Website
5. Set to Public (or Private if preferred)
6. Do NOT check "Add a README file" (we already have our files)
7. Click "Create repository"
8. Copy the repository URL, e.g.:
   https://github.com/YOUR-USERNAME/ketsy-store.git


STEP 4: Open Terminal and Navigate to Your Project
----------------------------------------------------
Mac/Linux:
  cd ~/Documents/Claude/Projects/KETSY\ STORE\ \(1\)

Windows (Git Bash):
  cd "C:/Users/YourName/Documents/Claude/Projects/KETSY STORE (1)"


STEP 5: Initialize Git and Push
---------------------------------
Run these commands one by one in your terminal:

  # Initialize git in the project folder
  git init

  # Add all project files
  git add .

  # Create your first commit
  git commit -m "Initial commit: Ketsy Store e-commerce website"

  # Connect to your GitHub repository (replace YOUR-USERNAME)
  git remote add origin https://github.com/YOUR-USERNAME/ketsy-store.git

  # Push to GitHub
  git branch -M main
  git push -u origin main

  ✅ Done! Your website is now on GitHub.


STEP 6: View Your Repository
------------------------------
Go to: https://github.com/YOUR-USERNAME/ketsy-store
You'll see all your project files there.


STEP 7: (Optional) Deploy Your Website FREE with GitHub Pages
--------------------------------------------------------------
1. Go to your repository on GitHub
2. Click "Settings" (top right area)
3. Scroll down to "Pages" in the left sidebar
4. Under "Source", select "Deploy from a branch"
5. Choose branch: "main", folder: "/ (root)"
6. Click Save
7. Wait ~2 minutes, then your site will be live at:
   https://YOUR-USERNAME.github.io/ketsy-store/

🌐 Your Ketsy Store website will be publicly accessible!


FUTURE UPDATES – How to Push Changes
--------------------------------------
Whenever you make changes to the website, run:

  git add .
  git commit -m "Update: describe what you changed"
  git push

Your GitHub repository and GitHub Pages site will update automatically.
